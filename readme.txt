== Description ==

Fast and easy WordPress plugin to insert Google Adsense to all your pages and posts with the possibility of excluding other users from use.

= Adsense Plus Ultra Features =

* Automatic Ads: insert Google Adsense code insede tag &lt;head&gt;
* Ads for single post: you can add you Google Adsense code before post content, insede post content (after second &lt;p&gt; tag), after post content
* Possibility to enable Adsense Plus Ultra plugin only for admin or for those who can manage plugins.
* Possibility to enable "no show ads" for logged users

= Adsense Plus Ultra Usage =

Once you have installed the plugin there are some options that you need to configure first in the plugin settings Tab.

* 1. Enable Adsense Plus Ultra only for admin or for those who can manage plugins
* 2. No show ads for logged users (easy way for avoid auto click)

Basic configuration on Ads Tab

* 1. Before proceeding it is always necessary to enable advertising
* 2. After enabling, you can enter the code provided by Google


== Installation ==

1. Go to the Add New plugins screen in your WordPress admin area
2. Click the upload tab
3. Browse for the plugin file (adsense-plus-ultra.zip)
4. Click Install Now and then activate the plugin
5. Now, go to the settings menu of the plugin and follow the instructions

== Frequently Asked Questions ==

= This plugin can be used with codes other than Google Adsense? =

No. Work only with Google Adsense code.
